THIS DIRECTORY CONTAINS THE FOLLOWING FILES:

* Feature multisite_custom_error-7.x-1.0.
  This is a copy of first version of the feature that provides simpletest scripts:
	- multisite_custom_error.test (QA test)
	- multisite_custom_error.extra.test (Test the Multisite Custom Error feature to ensure it works as intended.)

* In the folder Modules-Required are the modules required by the Feature.

* In the folder CodeReview is the feedback provided by the Coder module.


THESE ARE THE INSTALLATION STEPS:

1. Install and activate the required modules.

2. Install and activate the feature "multisite_custom_error".
